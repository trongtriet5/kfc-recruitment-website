# Logo Directory

Đặt logo của công ty vào thư mục này với tên file: `logo.png`

## Yêu cầu:
- Format: PNG (khuyến nghị) hoặc SVG
- Kích thước: Tối ưu cho hiển thị
  - Trang đăng nhập: Chiều cao khoảng 80-100px
  - Sidebar: Chiều cao khoảng 40px
- Nền: Trong suốt (transparent) hoặc màu trắng
- Tỷ lệ: Giữ nguyên tỷ lệ gốc

## Cách sử dụng:
1. Đặt file logo với tên `logo.png` vào thư mục này
2. Logo sẽ tự động hiển thị trên:
   - Trang đăng nhập (phía trên form)
   - Sidebar (góc trái trên cùng)

## Lưu ý:
- Nếu không có logo, hệ thống sẽ ẩn phần hiển thị logo và chỉ hiển thị text
- Đảm bảo file có kích thước hợp lý để tối ưu tốc độ tải trang
